Machine Learning Project

IPL Win Probability Predictor

A second-innings match-state predictor built from IPL ball-by-ball data using feature engineering, leakage-free evaluation, and interchangeable Logistic Regression and XGBoost pipelines.

View Repository Read Documentation

Project Snapshot

  • Predicts the chasing side's win probability in live IPL match situations
  • Supports Logistic Regression and tuned XGBoost pipelines
  • Uses match-level splits to prevent train-test leakage
  • Includes a Streamlit interface for interactive predictions

GitHub Pages can host this project page, but the interactive Streamlit app itself requires a Python runtime. To run the predictor live, deploy the app on Streamlit Community Cloud, Render, Railway, or another Python host.

How It Works

The training pipeline starts from raw IPL ball-by-ball data, isolates second-innings chase states, engineers contextual features, and evaluates models using match-level train, validation, and test splits.

archive/ipl_data.csv -> preprocess.py -> feature_engineering.py -> train_model.py -> app.py

Prediction Inputs
